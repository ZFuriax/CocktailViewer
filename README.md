# CocktailViewer

A lightweight Windows application for displaying images in a cocktail orientation, allowing everyone seated around a digital tabletop to view images without seeing them upside down.

**Platform:** Windows  
**Author:** Z  

---

## Download

Download the latest version from the [Releases](../../releases) page.

- `CocktailViewer-v1.0.0-win-x64.zip`

---

## Overview

CocktailViewer is designed for tabletop RPG sessions where players gather around a shared screen or digital table.

Instead of forcing some players to view images upside down, CocktailViewer displays the same image in multiple orientations simultaneously—so everyone at the table sees it correctly from their perspective.

---

## Features

### 🔄 Multi-Orientation Display
- Displays images rotated for multiple viewing angles
- Ideal for 2-sided or 4-sided table setups
- Ensures no player has to view content upside down.

### 🖼 Image Viewing
- Clean, distraction-free image display
- Supports common image formats
- Fast loading and smooth rendering

### 🎲 Built for TTRPGs
- Perfect for battlemaps, handouts, and visual aids
- Designed for in-session use
- Keeps players immersed without awkward repositioning

### ⚡ Lightweight & Fast
- Minimal system requirements
- Quick startup
- No unnecessary overhead

## Example Use Case

Place a monitor or TV flat on a table for your players.

CocktailViewer displays the image rotated in multiple directions so that:
- Players on each side of the table see it correctly
- No one needs to tilt their head or rotate the screen
